Performance Co-Pilot visualisation tools
for Apple(R) Mac OS X

pmchart is designed to produce stripcharts from Performance Co-Pilot (PCP) performance metrics fetched from either a live source (one or more hosts) or from one or more PCP archives.

pmtime is a graphical time controller utility that coordinates time updates and VCR-like playback for other utilities, like pmchart.

Performance Co-Pilot (PCP) is a framework and services to support system-level performance monitoring and performance management.

For more information and details on how to contribute to pmchart and the PCP project see the web pages at http://oss.sgi.com/projects/pcp/
