What's New in PCP GUI

This package provides the first open implementation of the original SGI pmchart utility. It offers all of the important functionality of the original tool and several interesting advances. It also offers a more modern user interface supporting Tabbed viewing, a toolbar, styles, gradient shading of widgets, and so on, making use of these now commonplace user interface components to provide the next step forward, while aiming to retain all of the usefulness of the original.

 

PCP GUI 1.0 - pmchart

  • Multiple Tabs

  • Support for simultaneous Live and Archive mode

  • Redesigned metric selector, allowing all aspects of charts to be setup (or edited) at once

  • Metric and instance name search and selection facility

  • Ability to vertically resize and temporarily collapse individual charts

  • Extended record functionality, allowing log size querying and creation of an Archive Tab at the end of the recording

  • Extended printing and image export support, with many image formats supported, optional background transparency and image scaling

  • Configurable toolbar for quick access to commonly used functionality

  • User interface configurability within the tool instead of external configuration files only

  • Native platform configuration mechanisms used for user interface Preferences

  • User interface support for creating and using color schemes (all aspects of the specification language now editable via user interface)

  • Support for MacOSX, X11 (Linux) and Windows platforms (Qt based)

     

Original SGI pmchart

  • VCR-like time control model

  • Vertically aligned charts, unified time axis

  • Separate time control process, keeping multiple clients updated with the same time position simultaneously

  • Source data from multiple hosts simultaneously

  • Source data from live hosts and archives using the same interface

  • View specification language, with multiple metrics, multiple charts, color schemes, and regular expressions on instance names

  • Record mode, with pmlogger(1) extensions and communication protocol

  • User interface configurability through X11 resources

  • Printing support, with export to PostScript and GIF format incorporated

  • Auto-scaling of Y-Axis metric values

  • Most aspects of the specification language editable via the user interface

  • Support for IRIX and Linux platforms (X11 based)

  • And much, much more...
